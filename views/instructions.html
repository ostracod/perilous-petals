Controls:

  • WASD or arrow keys = Move player
  • WASD or arrow keys + shift = Place or remove item
  • 0-9 keys = Select hotbar item
  • 0-9 keys + shift = Assign hotbar item
  • Enter key = Chat

Point values:

  • Planting seed = −1 point
  • Picking rose = +3 points
  • Picking lily = +5 points
  • Picking marigold = +8 points
  • Picking poisonous flower = −20 points
  • Destroying seed or sprout = −1 point
  • Placing or removing block = Free! :-)
